# 📚 Online Book Store SQL Project

## 📖 Project Overview

This project demonstrates practical SQL Server skills by designing, building, and querying an Online Book Store database. It includes database creation, CSV data import, and 50+ SQL queries covering business analysis and interview-oriented SQL problems.


## 🎯 Project Objectives

- Design a relational database.
- Import data using BULK INSERT.
- Solve real-world business problems using SQL.
- Strengthen SQL interview preparation through practical queries.

## 🗂️ Database Tables

- Books
- Customers
- Orders


## 📊 SQL Concepts Covered

- Database Creation
- Table Creation
- Primary Key & Foreign Key
- BULK INSERT
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- INNER JOIN
- LEFT JOIN
- Subqueries
- Correlated Subqueries
- COUNT(DISTINCT)
- TOP
- Date Functions
- COALESCE
- ROUND


## 📈 Business Questions Solved

- Top-selling books
- Revenue analysis
- Customer spending analysis
- Genre-wise sales
- Author-wise sales
- Monthly sales analysis
- Remaining stock calculation
- Books never ordered
- Customers without orders
- Top customers by spending
- 50+ SQL queries covering business scenarios and interview questions


## 🛠️ Tools Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- GitHub
- CSV Dataset

## 🚀 How to Run

1. Open SQL Server Management Studio.
2. Run `Online_Book_Store_SQL.sql`.
3. Import the CSV files using BULK INSERT.
4. Execute the SQL queries.
